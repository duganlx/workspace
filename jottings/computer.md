### 主板

##### Q: 主板基本情况说明?

因为有生产力需求，所以只选择 N 卡。那么对应主板有四种等级 X/Z/B/H，X 开头为最高级，搭配带有 X 结尾的 CPU；Z 开头为次旗舰，一般搭配带 K 结尾的 CPU；B 开头为中端主流，不支持 CPU 超频，主要搭配不带 K 结尾的 CPU。H 为入门级，不支持超频。于我而言，优先选择 B 开头的即可。

主板买一线品牌，如华硕、微星、技嘉。华硕的产品等级划分为 ROG MAXIMUS（旗舰）、ROG Strix（高端）、TUF GAMING（中端）、PRIME（入门）；微星的产品等级划分为 MEG（旗舰）、MPG（高端）、MAG（中端）、PRO（入门）；技嘉的产品等级划分为加强雕/超级雕（旗舰）、魔鹰/小雕（中端）、UD/DS3H（入门）。

[主板 B760M-MORTAR_WIFI_MAXWIFI 官方文档地址](https://download.msi.com/archive/mnu_exe/mb/B760MMORTAR_WIFI_MAXWIFI.pdf)

_内存规格_

```
4个 DDR5 内存插槽，支持高达 192GB
支持 1R 5600 MHz（通过 JEDEC 和 POR）
支持最高超频频率：
- 1DPC 1R 支持最高 7000+ MHz 内存
- 1DPC 2R 支持最高 6600+ MHz 内存
- 2DPC 1R 支持最高 6400+ MHz 内存
- 2DPC 2R 支持最高 5600+ MHz 内存
支持 Intel XMP 3.0 OC
支持双控制器双通道模式
支持非-ECC，非-缓存内存

解读：
主板支持单面内存模块（1R），最高时钟频率为 5600 MHz，这些规格是按照 JEDEC 标准定义的，并且可能需要通过重新启动或上电重置（POR, Power-On Reset）来配置；
支持最高超频频率：
- 若每个内存通道有一个单面的内存模块，则时钟频率可以达到 7000 MHz 或更高
- 若每个内存通过有一个双面的内存模块，则时钟频率可以达到 6600 Mhz 或更高
- 若每个内存通过有两个单面的内存模块，则时钟频率可以达到 6400 Mhz 或更高
- 若每个内存通过有两个双面的内存模块，则时钟频率可以达到 5600 Mhz 或更高
主板具有支持 Intel XMP 3.0 技术的功能，用户可以通过启用 XMP 来轻松地超频其内存模块，以实现更高的性能
主板具有两个独立的内存控制器，并且支持将内存模块配置为双通道模式
主板允许使用没有错误纠正功能和没有缓存的普通内存模块
```

说明：

- DPC 为 DIMM Per Channel 缩写，指主板与处理器之间的通信通道支持内存模块数量。
- 内存模块有时可以是单面的（1R, Single Rank）或双面的（2R, Dual Rank）。单面模块只在一个面上有内存芯片，而双面模块在两个面上都有内存芯片。通常来说，单面模块在一些情况下可能会有更好的性能。OC 为 Overclocking，即超频。
- 内存控制器是负责管理系统中内存模块的硬件组件。一般情况下，主板上会有一个或多个内存控制器。双控制器意味着两个独立的内存控制器，每个控制器负责一部分内存通道。
- 内存通道是主板与内存模块之间的物理连接。通过将内存模块分配到两个通道中，可以实现双通道模式。这种模式可以提高内存带宽，从而提高系统性能。在双通道模式下，数据可以同时通过两个通道传输，增加了内存访问的效率。

_扩展插槽_

```
2个 PCIe x16 插槽
- PCI_E1 插槽（来自CPU）
  - 支持 PCIe 5.0 x16
- PCI_E3 插槽（来自 B760 芯片组）
  - 支持 PCIe 4.0 x4
1个 PCIe x1 插槽
- PCI_E2 插槽（来自 B760 芯片组）
  - 支持 PCIe 3.0 x1

解读：
主板上有两个物理尺寸为 PCIe x16 的插槽，但是它们的实际带宽是不同的，一个是 PCIe 5.0 x16，另一个是 PCIe 4.0 x4
主板上有一个物理尺寸为 PCIe x1 的插槽，实际的带宽是 PCIe 3.0 x1
```

说明：

- PCIe 即为 PCI Express（外围组件快速互连），是一种高速串行扩展总线标准，通常用于连接显卡、固态硬盘以及采集卡和无线网卡等外设。
- PCIe 插槽中 x16、x4、x1 表示是物理尺寸，也表示通道数，通道数越多意味着带宽越高、插槽也更长；而 PCIe 5.0 x16、PCIe 4.0 x4、PCIe 3.0 x1 则是表示 PCIe 不同的规格（5.0/4.0/3.0）下实际占用的带宽（16/4/1 个通道）
- 每一代 PCIe 的速度都是上代的两倍。PCIe 3.0 的数据传输速率为每秒 8 GB，而 PCIe 4.0 为 16 GT/秒，PCIe 5.0 为 32 GT/秒。（比特率的度量单位为千兆传输速率，表示的是编码前的理论最大速度——现实中的速度可能会更慢。）（来源：[什么是 PCIe 4.0 和 5.0？| intel](https://www.intel.cn/content/www/cn/zh/gaming/resources/what-is-pcie-4-and-why-does-it-matter.html)）

_M.2 固态硬盘接口_

```

- M2_1 接口（来自CPU）

  - 支持 2242/ 2260/ 2280 存储设备
- M2_2 接口（来自B760芯片组）

解读：
M.2 插槽上安装长度为 42mm、60mm 或 80mm 的 M.2 固态硬盘
```

说明

- M.2 是一种用于连接固态硬盘（SSD）和其他存储设备的接口标准。该接口的插槽端有三种设计，一种支持 B key（B 键）的插槽，短的一段在左边，采用 6pin 设计，采用 B key 的 M.2 插槽比较低端，只能走 SATA 或 PCIe x2 的通道。另一种是支持 M key（M 键）的插槽，短的一段在右边，采用 5pin 设计。而采用 M key 的插槽就要高端些，可走 PCIe x4 的通道，也是现在主板采用的主流接口方式。还有第三种就是 B&M key 的插槽，这种接口是三段式设计的，既可以插 B key 的插槽，也可以插 M key 的插槽，但是这种接口所采用的通道只能是 SATA 或 CPIe×2，所以 B&M key 的固态读写速度相比 M key 的固态要慢上不少。（来源：[全网最全，尔英电脑主板全接口解析，只需收藏这一篇就够啦！](https://www.erying.cc/news-detail/id-84.html)）
- 2242/2260/2280 表示 M.2 存储设备的长度。分别表示的长度为 42mm/60mm/80mm

频率 & 位宽

双通道可以让位宽翻倍，两个内存好，插 2，4 槽，选 D-Type 布线，无残线导致的电气损失。

imc 内存控制器（CPU 中的北桥?）

在其他因素不构成瓶颈的情况下，你购买的内存，其商品界面标注的频率为该内存的保底值，代表该内存厂商保证小于等于这个频率可以毫无压力运行，你可以自己手动尝试更高频率

在主板能不超频的情况下，内存频率的下限由内存本身决定，内存频率的上限由 CPU 默认内存频率与内存体质较低的那个决定

在对内存超频的情况下，内存频率将无视 CPU 的默认频率，此时内存频率的下限由内存本身决定，内存频率上限由内存体质、IMC 性能/体质、主板布线与电气性能这三者最低的值决定

_Q: ddr4 中，CJR 和 DJR 是什么意思?_

CJR（C-Die Junior）： C-Die 是一种由 Micron 制造的 DDR4 内存芯片。CJR 通常是 Micron C-Die 芯片的一种子型号或新一代，通常被认为是性能较好的内存芯片。CJR 芯片被用于制造各种品牌的 DDR4 内存模块。

D-Die 是由 Samsung 制造的 DDR4 内存芯片。DJR 则是 Samsung D-Die 芯片的一个子型号或新一代。类似于 C-Die，DJR 芯片通常被认为具有良好的性能，并用于制造高性能 DDR4 内存模块。

### 固态

固态硬盘核心存储介质是闪存，具体分为 SLC、MLC、TLC，其他差别在于每个单元存储的比特数不同。

固态缓存：DRAM 动态随机存取存储器; SLC Cache SLC 模拟高速缓存

浮栅晶体管构成分：SLC(1bit); MLC(2bit); TLC(3bit)

NAND 颗粒里边包含长方体组成的浮栅晶体管，缓存减少 NAND 颗粒的读写。

TBW：TB write 指 SSD 生命周期内可以写入的总字节数。使用寿命计算公式为 `年数 = (固态硬盘的TWB(TB) x 1024) / (365天 x 1天写入量GB)`

DWPD 确定用户在保修期内每天可以重写驱动器的次数，公式`DWPD = TWB / 保修期天数`

总线（SATA, PCIe, SAS）、协议（AHCI, NVMe, SCSI）、接口（SATA, mSATA, SATA Express, PCIe, M.2）

dB 噪声的标准计量单位: 分贝；dBA 是根据人类听觉属性设计的一种噪声测量单位。它采用 A 权重滤波器对噪声信号加权，以反映人类听觉系统对不同声压级频率的响应。在测量噪声时，通常使用 dBA 作为参照标准，以便更准确地衡量对人类听觉系统的影响。dB 参考值: 安静的图书馆(30-40dB); 室内对话(50-60dB); 市区交通(70-80dB); 现场演唱会(90dB-100dB)

### 屏幕

LCD 屏幕: VA 屏幕、IPSP 屏幕、TN 屏幕

- 一开屏幕整个背光层都需要打开
- 优点：寿命长
- 缺点：对比度低，存在漏光，响应时间长，厚度厚，不好弯曲，功耗高；理论分辨率达不到理论清晰度
- 对眼：LCD 蓝光 > OLED 蓝光；OLED 有 PWM 调光 存在频闪

| 指标 \ 屏幕 | VA 屏幕          | IPS 屏幕   | TN 屏幕      |
| ----------- | ---------------- | ---------- | ------------ |
| 对比度      | 高               | 一般       | 很低         |
| 可视角度    | 一般             | 广         | 较差         |
| 响应速度    | 较慢             | 适中       | 最快         |
| 适合用户    | 电影,3A 单机用户 | 大多数用户 | FPS 竞技游戏 |

OLED 有机自发光二极管

- 每个像素点可以单独控制开关，可以做到`息屏提醒`(部分点亮)
- 优点：对比度高，没有漏光，响应时间短，厚度薄，可弯曲，功耗低
- 缺点：寿命短；存在烧频；
- 调光采用 PWM，会有屏闪，对眼睛不友好。

| 指标\ 档次 | A 档       | B 档          | C 档     |
| ---------- | ---------- | ------------- | -------- |
| 亮度       | ≥400nit    | ≈350nit       | <250nit  |
| 对比度     | ≥1200:1    | ≥1000:1       | <900:1   |
| 色准       | ΔE<3       | ΔE<5          | ΔE≥5     |
| 响应速度   | <3ms       | <5ms          | ≥10ms    |
| 色域覆盖   | ≥95%DCI-P3 | 95%~100% sRGB | <90%sRGB |
| 色彩均匀度 | ≤ΔE3       | ≤ΔE5          | ΔE5~10   |
| 亮度均匀度 | ≤5%        | ≤15%          | ≥20%     |
| 色温       | 6500K      | 6700~7000K    | > 7000K  |

OLED 采用 Pentitle 原因：像素点种植成本、屏幕老化问题

micro-LED：将 OLED 的有机自发光二极管替换成无机材料的 LED 灯珠的屏幕

PPI

- 公式 sqrt(x^2 + y^2) / 屏幕尺寸
- 300 够用；360 清晰；420 锐利
- 只适用 RGB 排列的 LCD 屏幕，不适用 OLED
- OLED 下 PPI = sqrt(x^2 + y^2) / 屏幕尺寸 x 81.65%，即 Pentitle 排列的清晰度 = RGB 排列的 81.65%

色域

- 100% sRGB ≈ 79.6% DCI-P3
- 100% sRGB 当下能提供最好的体验

色准：色彩实际颜色和显示器显示的颜色的差距

- ΔE 越小代表色彩偏离越小，色准越高
- ΔE ≤ 1.5 色准极好 S 级 物理意义上几乎不存在色偏
- ΔE 1.5 - 3 非常优秀 A 级 专业人士用肉眼也很难看出存在色偏
- ΔE 3 - 5 优秀 B 级 日常使用几乎不会有影响
- ΔE 5 - 8 普通 C 级 色彩敏感的人枯杨看出有偏色
- ΔE > 8 很差 D 级 大部分人都能看出些微的偏色

更优先参看指标：分辨率、刷新率、响应时间、面板类型

色深

- FRC 通过类似 PWM，色 1 和色 2 进行交替闪烁出现色 1.5
- 6bit 平滑色彩过度会出现断层 不推荐
- 6 抖 8 细看会有静态噪点 凑合
- 8bit 够用不贵 性价比高 推荐
